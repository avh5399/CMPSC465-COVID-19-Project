Infection Transmission Rate Forecast Simulation of COVID-19

Shivran Muralidhar, Vivek Gopalan, Harsh Maniar, Devkumar Ojha, Ameya Hampihallikar



Project Description

This project will build a sophisticated system that will forecast the spread and transmission rate of COVID-19 on a given city or state based on established data models provided by the CDC, WHO, and other organizations.

This project will allow different bodies of government to understand the spread and flow of COVID-19 and help them make wise decisions when estimating the time periods on lockdowns.

Updates

Edited (04/12/2020) | Combined Work: >70 hours

The team was able to utilize raw input sets of data to find the mean and standard deviations of the number of cases in respective states and counties, and build a Poisson process model on python using NumPy, Pandas, and Statistics libraries using PyCharm environment. The team uses Google Docs to edit and review processes and methods. The team uses CodeShare.io to collaborate and build python simulation code. The expected delivery of the derived data and results from the project and results is April 30, 2020.

Edited (04/20/2020) | Combined Work: >100 hours

Focus was shifted to the area of Brooklyn in New york to simulate out program. We consolidated the data using our statisitical data model and filterd it based on the brooklyn area zipcodes. Using matplotlib, a heatmap was generated for the time period of one month. (31-03-20 to 25-04-20). Using the heat map we have created a visual representation to show the predicted rise in COVID-19 cases. Darker the red circles represent the increase in density of cases for the corresponding zipcode.
